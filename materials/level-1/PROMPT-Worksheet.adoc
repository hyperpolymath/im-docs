a|

|**O**bjectivity
|_[0.0-1.0]_
a|
* What biases might exist?
* Are they disclosed?
* Funding sources transparent?
* Ideological leanings acknowledged?

|**M**ethods
|_[0.0-1.0]_
a|

* Sample size adequate?
* Peer-reviewed?
* Replicable?
* Methodology sound?

|**P**erspective
|_[0.0-1.0]_
a|
* Whose voices are included?
* Who is missing?
* Affected communities consulted?
* Dissenting views represented?

|**T**imeliness
|_[0.0-1.0]_
a|
* When was this published?
* Is there more recent evidence?
* Has this been superseded?
